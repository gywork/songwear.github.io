GitHub

Github 全球最大的基友平台。国内还有一个 Gitcafe (2016年3月被CODING收购)，都是代码托管平台，当然后者和前者的差距还有很大，后面我们会提到 Gitcafe 。请不要使用 QQ 邮箱注册 Github，至于理由，我会在之后的文章中说明。至于如何注册以及验证邮箱，我就不再赘述，翻译工具已经很强大了。

NodeJS 环境

Hexo 是一个基于 Node.js 的静态博客程序。因此我们需要 NodeJS 来安装 Hexo。安装步骤没什么好说的，官网直接下载安装程序即可，也 无需 配置环境变量。(nodejs中已经包含npm无需再次下载)
